# AI Personal Life OS

An AI-powered personal assistant that understands context, manages tasks,
automates workflows, and maintains long-term memory.

## Core Capabilities
- Natural language task creation
- Context-aware reasoning
- Long-term memory using vector search
- Task automation and scheduling
- Modular, extensible architecture

## Project Structure
This repository follows a modular architecture separating:
- API layer
- Core AI reasoning
- Memory systems
- Automation and schedulers
- External services
- Monitoring and observability

## Tech Stack
- Python
- FastAPI
- LLMs (GenAI)
- Vector Databases
- Docker
- CI/CD (planned)
